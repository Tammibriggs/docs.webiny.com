In this tutorial, we will learn how to manage the general settings of a page. As an example, we will update the **Title** of the **Welcome to Webiny** example project.

<Video src={createContent} controls={true}/>

## General Settings
- **Title** : The page title.
- **Path** : The path for accessing the page.
- **Snippet** : The meta description of the page.
- **Layout** : The structure of the page.
- **Tags** : For filtering pages.
- **Page Image** : The meta image of the page. 

## Manage General Settings
1. From the **Side Menu**, Click **Page Builder** > **Pages**.  
2. Select **Welcome to Webiny** page.
3. In the page preview, Click the **EDIT CONTENT MODEL** icon (✏️).

   ✔️ The page editor opens. 
4. Click the **Settings** icon (⚙️).

   ✔️ The Page Settings screen opens.
5. In the **Title** textbox, update the **Welcome to Webiny** text to **Welcome**.

   **Note**: Similarly, you can update **Path**, **Snippet**, **Layout**, **Tags**, and **Page Image**.
6. Click **SAVE SETTINGS**.

   ✔️ The page editor opens.
7. Click **PUBLISHE CHANGES**.

   ✔️ The Publish page screen opens.
8. Click **CONFIRM**.
